# Captures d'écran

Ce dossier accueille les captures utilisées dans le `README.md`.
Remplace ces placeholders par tes propres captures, en gardant les mêmes noms de fichiers :

| Fichier attendu | Ce qu'il doit montrer |
|---|---|
| `01-workflow-canvas.png` | Le canvas du workflow ouvert dans n8n (tous les nœuds connectés). |
| `02-notion-database.png` | La base Notion avec quelques offres enregistrées. |
| `03-credentials.png` | La liste des credentials n8n (Gemini API, Notion, Gmail) bien nommés. |
| `04-execution-success.png` | Une exécution réussie (coche verte sur chaque nœud). |
| `05-email-notification.png` | L'e-mail de notification récapitulatif reçu. |

> Astuce : sur n8n, un clic droit sur le canvas → *Download* exporte une image propre du workflow.
